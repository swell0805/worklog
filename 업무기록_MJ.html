📅 Outlook 캘린더 가져오기

Outlook web에서 캘린더를 게시(Publish)하면 받는 ICS URL을 입력하세요.

📖 Outlook web에서 ICS URL 받는 법
  1. outlook.office.com 접속 → 회사 계정 로그인
  2. 우측 상단 ⚙ 설정 클릭 → 하단 모든 Outlook 설정 보기
  3. 좌측 캘린더 → 공유 캘린더(Shared calendars)
  4. 캘린더 게시(Publish a calendar) 섹션:
    • 캘린더: 기본 "캘린더"
    • 권한: 모든 세부 정보 보기 가능
    • 게시(Publish) 버튼
  5. 나타난 두 링크 중 ICS 링크(.ics로 끝남) 복사 → 아래 붙여넣기
  6. ⚠️ 게시 메뉴가 회색이거나 안 보이면 회사 IT가 차단한 상태

⚠️ 보안 참고: ICS는 브라우저 보안(CORS)상 직접 못 읽어 무료 프록시(corsproxy.io)를 거칩니다. 캘린더 내용이 프록시 서버를 일시 통과합니다.

Azure AD에서 받은 Client ID를 입력하세요. (1회만 — 모든 기기에서 같은 ID 사용)
아래 "📖 Azure AD 설정 가이드 보기"로 받는 법 확인 가능

📖 Azure AD 설정 가이드 보기
  1. Azure Portal 접속 (회사 Microsoft 계정으로 로그인)
  2. 상단 검색창에 App registrations → 클릭
  3. + New registration 클릭
  4. 이름: 업무기록 worklog (자유)
  5. 지원 계정 유형: Accounts in any organizational directory and personal Microsoft accounts
  6. Redirect URI: Single-page application (SPA) 선택, URL은 (앱 URL)
  7. Register 클릭
  8. 등록 후 표시되는 Application (client) ID 값 복사 → 아래 입력
  9. 좌측 API permissions → + Add a permission → Microsoft Graph → Delegated → Calendars.Read 체크 → Add
  10. 회사 계정이면 Grant admin consent 버튼이 보일 수 있음 — IT 도움 필요

Microsoft 계정으로 로그인하면 캘린더 일정을 가져올 수 있습니다.
